# Kitchen Picks

## サイト概要
「キッチン便利グッズ」をレビュー・共有できるコミュニティサイトです。
### サイトテーマ
忙しい毎日の中で家事の負担を減らすためのアイテムを、
ユーザー同士で共有し合えるレビュー型サービスです。

​
### テーマを選んだ理由
私は普段からキッチン便利グッズを使うことが多く、
「これ便利！」と思ったアイテムを誰かにおすすめしたいと感じることがよくあります。
しかし、実際に使ってみたリアルなレビューをまとめて見られる場所が少なく、
購入前に比較できるサービスがあれば便利だと考えました。
また、家事や料理に時間をかけられない人にとって、
“時短できるアイテム”の情報は大きな価値があります。
そこで、ユーザーが実体験を共有し合えるレビューサイトを作ることで、
家事の負担を減らす手助けができると考え、このテーマを選びました。

### ターゲットユーザ
- 家事の時間を短縮したい人
- キッチン便利グッズが好きな人
- 新生活を始める人
- どのアイテムを買うか迷っている人
- 100均・時短グッズの情報を集めたい人


​
### 主な利用シーン
- 新しいキッチングッズを買う前にレビューを確認したいとき
- 実際に使って良かったアイテムを共有したいとき
- 「時短」「収納」「100均」などのカテゴリから探したいとき
